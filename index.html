Step 0 Install Chocolatey (Package Manager) üç´

Chocolatey makes installing software on Windows much easier.

1. Open PowerShell as Administrator:

  • Press Windows Key
  • Type: PowerShell
  • Right-click on "Windows PowerShell"
  • Click "Run as administrator"
  • Click "Yes" if asked for permission

2. Fix Execution Policy (if you get an error):

Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
  • Type Y and press Enter when asked

3. Install Chocolatey:

Copy and paste this command:

Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
  • Press Enter
  • Wait for installation (1-2 minutes)

4. Verify installation:

choco --version
You should see something like: 2.2.2
Step 1 Install Node.js, Git, and VS Code ‚¨áÔ∏è

Install all three at once (in PowerShell as Administrator):

choco install nodejs-lts git vscode -y
This will install:
  • ‚úÖ Node.js (JavaScript runtime)
  • ‚úÖ Git (version control)
  • ‚úÖ VS Code (code editor)

Wait for installation (5-10 minutes)

Step 2 Verify Installations ‚úîÔ∏è

Close and reopen PowerShell, then check:

1. Node.js:

node --version
Should show: v20.10.0 (or similar)

2. npm (comes with Node.js):

npm --version
Should show: 10.2.3 (or similar)

3. Git:

git --version
Should show: git version 2.42.0 (or similar)

4. VS Code:

code --version
Should show version number
Step 3 Download the Newspaper Archiver üì•

1. Open Command Prompt (or PowerShell):

  • Press Windows Key + R
  • Type: cmd (or powershell)
  • Press Enter

2. Navigate to your desired folder:

cd your-folder-name

(Replace "your-folder-name" with where you want to install. Example: cd Desktop or cd Documents)

3. Download the project:

git clone https://github.com/MihalKazi/newspaper-archiever.git

4. Enter the project folder:

cd newspaper-archiever
Step 4 Open Project in VS Code üíª

From VS Code:

  1. Open VS Code
  2. Click "File" ‚Üí "Open Folder"
  3. Navigate to Desktop\newspaper-archiever
  4. Click "Select Folder"
Step 5 Install Dependencies üì¶

1. Open the integrated terminal:

  • Press Ctrl + ` (backtick key, usually below Escape)
  • Or: Menu ‚Üí "Terminal" ‚Üí "New Terminal"

2. Install npm packages:

Wait for it to finish (3-5 minutes, downloads ~200MB)

3. Install Playwright browsers:

npx playwright install
This downloads the browser (takes 12-15 minutes)
Step 6 Start the Application üöÄ

In VS Code terminal:

You'll see:
Server running on http://localhost:3000

Open your web browser:

  • Open Chrome, Firefox, or Edge
  • Go to: http://localhost:3000
üéâ You're ready!
Step 7 How to Use üìù
  1. Copy an article URL from any news website
    Example: https://www.thedailystar.net/business/news/article-12345
  2. Paste it into the text box on the website
  3. Click "Archive This Article"
  4. Wait while it downloads (30 seconds to 2 minutes)
  5. Find your archived article:
    • Go to: newspaper-archiever\archives\
    • Open the folder with the website name
    • Browse by date to find your article
How to Stop the Application üõë

In VS Code terminal:

  1. Press Ctrl + C
  2. Type Y when asked
  3. Press Enter
How to Start Again Later üîÑ

Option 1 - Using VS Code:

  1. Open VS Code
  2. Open the newspaper-archiever folder
  3. Open terminal (Ctrl + `)
  4. Type: npm start
  5. Open browser: http://localhost:3000

Step 0 Install Homebrew (Package Manager) üç∫

Homebrew makes installing software on macOS much easier.

1. Open Terminal:

  • Press Command (‚åò) + Space
  • Type: Terminal
  • Press Enter

2. Install Homebrew:

Copy and paste this command:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  • Press Enter
  • Enter your Mac password when prompted (you won't see it typing)
  • Press Enter again to confirm
  • Wait for installation (3-5 minutes)

3. Add Homebrew to your PATH (if needed):

After installation, you may see instructions to run these commands:

echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile eval "$(/opt/homebrew/bin/brew shellenv)"
Note: Copy and run these commands if Terminal tells you to.

4. Verify installation:

brew --version
You should see something like: Homebrew 4.x.x
